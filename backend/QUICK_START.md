# 🚀 Quick Start - Bypass PowerShell Issue

## ✅ EASIEST SOLUTION: Use CMD Instead

**Don't use PowerShell, use Command Prompt (CMD):**

1. Press `Windows Key + R`
2. Type: `cmd`
3. Press Enter
4. Navigate to backend:
```cmd
cd D:\AiSensyWebApp\backend
```
5. Run:
```cmd
npm run dev
```

**CMD doesn't have execution policy issues!** ✅

---

## ✅ ALTERNATIVE: Use Batch File

**I created batch files for you:**

1. Go to: `D:\AiSensyWebApp\backend\`
2. Double-click: `START_SERVER_DEV.bat`
3. Server will start automatically!

---

## ✅ ALTERNATIVE: Use npm.cmd in PowerShell

**In PowerShell, use:**
```powershell
npm.cmd run dev
```

Instead of:
```powershell
npm run dev
```
